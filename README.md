# Impact-of-Population-Growth-on-Environmental-Degradation
My main goal is to create a mathematical model using differential equations to describe the impact of population growth on environmental degradation in Kenya, with results visualized using R.
This idea came from the fact that I really struggled, still do, learning differential equations and I recently did an assignment for my Sustainable Resource Management class ENV2001 titled 'IMPACT OF POPULATION GROWTH AND HUMAN-INDUCED DEVELOPMENT ACTIVITIES ON ENVIRONMENTAL DEGRADATION:SUSTAINABLE APPROACHES AGAINST THE VICE GIVING RELEVANT EXAMPLES FROM MY COUNTRY' which took a lot of time and effort. 
I am excited on being able to combine these 2 courses.
This is also my first math centered project so I am very excited.

## Steps I will be taking
1. Find a Dataset that shows Impact of Population Growth on Environmental Degradation in Kenya
2. Solve differential equations by hand to get answers from the dataset
3. Visualize and Analyse my findings in R.
